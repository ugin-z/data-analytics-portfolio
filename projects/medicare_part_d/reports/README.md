# Reports

This directory contains the final analytical outputs of the project.
The files in this folder summarize key findings, insights, and methodology,
and are intended to be read without referring to the underlying notebooks.

## executive_summary.md
A high-level summary of the project, highlighting the main analytical
questions, key results, and overall conclusions.
Intended for readers who want a concise overview.

## key_findings.md
A structured list of the main analytical findings, organized by analytical
dimension (drug-level, prescriber-level, and drugâ€“prescriber interactions).
Provides supporting evidence for the conclusions presented in the executive summary.

## methodology.md
A brief description of the data sources, data architecture, and analytical
approach used in the project.
Intended to provide transparency and support the credibility of the findings.

## Reading Order
1. Start with 'executive_summary.md' for a quick overview.
2. Read 'key_findings.md' for detailed results and supporting evidence.
3. Refer to 'methodology.md' for background on data sources and analytical design.
